Github的使用(github.com)
	为客户提供免费/收费的的公共Git仓库托管服务，是全球最大的开源项目集散地。

企业中做多人协作的项目使用Git&Github的流程：
(1)项目发起人在github.com上创建一个项目仓库――公共仓库；并创建一些初始化文件，例如：
	README.md    项目说明文件
	LICENSE.txt  使用许可说明文件
	.gitignore   仓库忽略文件列表
(2)项目组员在自己的电脑上安装Git软件，克隆公共仓库到本地
	git clone 远程仓库地址
(3)每天早上项目组员从领导处领取当日任务，在本地仓库创建/修改/删除文件，添加到暂存区，提交到本地仓库
	git add .
	git commit -m ""
(4)晚上下班时项目组员把本地仓库中的内容推送到原始仓库
	git push origin
(5)第二天早上上班时所有队员从公共仓库拉取昨日的所有提交
	git pull origin


Github.com的使用步骤：
(1)准备一个邮箱
(2)注册一个Github账号，需要进行邮箱验证
(3)登陆Github账号，创建自己的项目仓库
(4)(可选)邀请其他人成为自己项目的“合作开发者”
(5)维护项目仓库中的文件和历史记录
